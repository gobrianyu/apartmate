# Apartmate

> **Building tighter, more reliable communities for off-campus students.**

Apartmate is a mobile-native solution designed to combat the "loneliness epidemic" in off-campus housing. By facilitating low-pressure, asynchronous social interactions, Apartmate helps students connect with their neighbors through shared interests, resource lending, and community events.

*View the website [here](https://apartmate-inky.vercel.app/), or at https://apartmate-inky.vercel.app/.*

## Key Features

- **Async Messaging:** Connect with neighbors on your own time without the pressure of instant replies or awkward face-to-face introductions.
- **Resource Sharing:** Borrow everyday items (like baking soda or a vacuum) from neighbors who are happy to help, with automated lending tracking.
- **Interest Groups:** Join building-specific groups based on your major, hobbies, or year to find your tribe asynchronously.
- **Building Integration:** Verified residents access ensures a safe, reliable, and exclusive community environment for University of Washington students.


## Tech Stack

- **Frontend:** React Native / React
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Backend:** Firebase (Auth & Firestore)
- **Language:** TypeScript


## Design Process

Our team followed a rigorous human-centered design process over the course of the quarter:

1.  **User Research:** Conducted surveys (n=28) and 3 semi-structured interviews with UW students to uncover emotional barriers to spontaneous connection.
2.  **Key Findings:** Identified that 85% of students feel isolated, primarily due to a "fear of rejection" in physical common areas.
3.  **Ideation:** Brainstormed solutions focused on reducing social friction through asynchronous, mobile-first interactions.
4.  **Prototyping:** Validated core user flows using paper prototypes and low-fidelity models before moving to high-fidelity design.
5.  **Final Pitch:** Presented our journey and solution to peers and mentors, focusing on user-centric community building.

## For Developers: Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/gobrianyu/apartmate.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
